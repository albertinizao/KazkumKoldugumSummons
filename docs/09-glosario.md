# Glosario

## Términos principales

**Criatura base**
Ficha original cargada desde JSON, sin reglas fijas aplicadas.

**Criatura final / resuelta**
Ficha resultante de aplicar Augment Summoning, plantilla y Deep Guardian cuando corresponda.

**Grupo activo**
Agrupación de instancias que comparten la misma criatura final.

**Instancia**
Una criatura concreta dentro de un grupo, con PG independientes.

**PG**
Puntos de golpe.

**CA**
Clase de armadura.

**TS**
Tirada de salvación.

**Usos diarios**
Contador persistido de usos disponibles para invocar.

**Últimas usadas**
Lista corta de accesos rápidos ordenada por uso reciente.

**Más usadas**
Lista corta de accesos rápidos ordenada por frecuencia.

**Asistente de invocación**
Flujo guiado que recomienda criatura y plantilla a partir de un árbol de decisiones.
